uses no library because it uses L298N motor driver 
electronic components needed: Arduino Uno R3 or Arduino Mega, 4 dc motors, 2 HW 870 proximity sensors, 7.4v lithium ion battery(batteries can vary but nothing lower than this)
I saw some other versions that used the l293D but I decided to do my own circuit that used simpler components and still created the same end result 
The chassis I used can be found on makerworld but designing one is also fun
